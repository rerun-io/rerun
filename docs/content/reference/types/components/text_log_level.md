title: "TextLogLevel"
---

The severity level of a text log message.

Recommended to be one of:
* `"CRITICAL"`
* `"ERROR"`
* `"WARN"`
* `"INFO"`
* `"DEBUG"`
* `"TRACE"`

* value: [`Utf8`](../datatypes/utf8.md)
