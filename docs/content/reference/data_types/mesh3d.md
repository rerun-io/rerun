A 3D triangle mesh as specified by its per-mesh and per-vertex properties.

## Components and APIs

Required components:
* `position3d`

Recommended components:
* `mesh_properties`
* `vector3d`

Optional components:
* `color`
* `material`
* `class_id`
* `instance_key`
