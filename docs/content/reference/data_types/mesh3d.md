A 3D triangle mesh as specified by its per-mesh and per-vertex properties.

## Components

**Required**: `Position3D`

**Recommended**: `MeshProperties`, `Vector3D`

**Optional**: `Color`, `Material`, `ClassId`, `InstanceKey`
