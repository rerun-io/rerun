---
title: "Points2D"
---

A 2D point cloud with positions and optional colors, radii, labels, etc.

## Components

**Required**: `Position2D`

**Recommended**: `Radius`, `Color`

**Optional**: `Text`, `DrawOrder`, `ClassId`, `KeypointId`, `InstanceKey`
