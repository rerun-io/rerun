A batch of 2d boxes with half-extents and optional center, rotations, rotations, colors etc.

## Components

**Required**: `half_sizes2d`

**Recommended**: `position2d`, `color`

**Optional**: `radius`, `text`, `draw_order`, `class_id`, `instance_key`
