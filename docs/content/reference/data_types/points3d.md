---
title: "Points3D"
---

A 3D point cloud with positions and optional colors, radii, labels, etc.

## Components

**Required**: `Position3D`

**Recommended**: `Radius`, `Color`

**Optional**: `Text`, `ClassId`, `KeypointId`, `InstanceKey`
