Log a double-precision scalar that will be visualized as a timeseries plot.

The current simulation time will be used for the time/X-axis, hence scalars
cannot be timeless!

## Components and APIs

Required:
* `scalar`

Recommended:
* `radius`
* `color`

Optional:
* `text`
* `scalar_scattering`
