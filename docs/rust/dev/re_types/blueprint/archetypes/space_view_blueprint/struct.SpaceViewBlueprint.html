Expand description

Archetype: The top-level description of the Viewport.

Fields§

§class_identifier: SpaceViewClass

The class of the view.

§display_name: Option<Name>

The name of the view.

§space_origin: Option<SpaceViewOrigin>

The “anchor point” of this space view.

The transform at this path forms the reference point for all scene->world transforms in this space view. I.e. the position of this entity path in space forms the origin of the coordinate system in this space view. Furthermore, this is the primary indicator for heuristics on what entities we show in this space view.

§entities_determined_by_user: Option<EntitiesDeterminedByUser>

True if the user is has added entities themselves. False otherwise.

§contents: Option<IncludedQueries>

BlueprintIds of the DataQuerys that make up this SpaceView.

It determines which entities are part of the spaceview.

§visible: Option<Visible>

Whether this space view is visible.

Defaults to true if not specified.
