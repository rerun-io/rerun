Expand description

Returns a new ClampedZip2x2 iterator.

The number of elements in a clamped zip iterator corresponds to the number of elements in the shortest of its required iterators (r0, r1).

Optional iterators (o0, o1) will repeat their latest values if they happen to be too short to be zipped with the shortest of the required iterators.

If an optional iterator is not only too short but actually empty, its associated default function (o0_default_fn, o1_default_fn) will be executed and the resulting value repeated as necessary.
