pub fn track_implicit_dep( &self, pkg_name: &str, files_to_watch: &mut HashSet<PathBuf>, )

Tracks an implicit dependency of the given name.

This will generate all the appropriate cargo:rerun-if-changed clauses so that package pkg_name as well as all of it direct and indirect dependencies are properly tracked whether they are remote, in-workspace, or locally patched.
