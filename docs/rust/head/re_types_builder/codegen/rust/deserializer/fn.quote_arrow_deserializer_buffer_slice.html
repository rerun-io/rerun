This generates code that deserializes a runtime Arrow payload into the specified obj, taking Arrow-transparency into account.

It contains additional performance optimizations based on the inner-type being a non-nullable primitive allowing us to map directly to slices rather than iterating. The ability to use this optimization is determined by should_optimize_buffer_slice_deserialize.

There is a 1:1 relationship between quote_arrow_deserializer_buffer_slice and Loggable::from_arrow:
