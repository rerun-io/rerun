struct PyComponentColumnDescriptor(ComponentColumnDescriptor);
Expand description

The descriptor of a component column.

Component columns contain the data for a specific component of an entity.

Column descriptors are used to describe the columns in a [Schema][rerun.dataframe.Schema]. They are read-only. To select a component column, use [ComponentColumnSelector][rerun.dataframe.ComponentColumnSelector].
